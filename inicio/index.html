BLOG DEL EQUIPO 6
JenkinsAbuser

¡Bienvenidos a nuestro blog! En esta sesión 04 estamos desarrollando un proyecto práctico usando el framework Hugo. Hugo nos permite crear sitios web estáticos de manera rápida, eficiente y profesional, con la posibilidad de personalizar el diseño y la estructura según nuestras necesidades.

El objetivo de este blog es documentar nuestro trabajo en el equipo HMIS, compartir conocimientos sobre desarrollo web estático y mostrar cómo se pueden combinar herramientas modernas de despliegue y hosting.

En esta primera fase estamos configurando la página de inicio para que sirva como introducción clara al proyecto. Queremos que los visitantes comprendan de inmediato el propósito del blog, nuestra metodología y cómo navegar por los contenidos principales.

Estamos utilizando este blog para:

  • Explicar cómo funciona Hugo y cómo estructuramos un sitio web estático.
  • Practicar el despliegue en GitHub Pages y en Microsoft Azure como sitios web estáticos.
  • Documentar paso a paso la evolución del proyecto del equipo HMIS.
  • Crear una referencia práctica para futuros desarrollos web en equipo.

Además, en este blog encontrarás secciones dedicadas a los desarrolladores del equipo, posts con detalles técnicos y explicaciones de cada sesión de trabajo. Nuestro objetivo es que, aunque estés llegando por primera vez, tengas toda la información necesaria para entender el proyecto y explorar los contenidos relacionados.

Puedes comenzar viendo nuestros próximos Posts, donde documentamos cada avance, problemas que hemos resuelto y aprendizajes clave. También encontrarás información sobre nuestro equipo en la sección de Desarrolladores, así como contacto y recursos adicionales.

Estamos convencidos de que un buen blog no solo documenta, sino que también inspira y guía a otros desarrolladores. Por eso, cada entrada está pensada para aportar valor, claridad y ejemplos prácticos que puedan ser replicados en proyectos propios.

¡Esperamos que disfrutes explorando JenkinsAbuser y aprendas tanto como nosotros mientras lo desarrollamos!
